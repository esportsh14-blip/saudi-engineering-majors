# Engineering Majors · Saudi Arabia

A bilingual (English / العربية) website to help students explore engineering majors offered in Saudi Arabia and discover the best universities for each — aligned with Vision 2030.

**Live site:** https://esportsh14-blip.github.io/saudi-engineering-majors/

## Features

- 13 engineering majors with descriptions, demand levels, and career sectors
- Best Saudi universities for each major (KFUPM, KSU, KAU, KAUST, Alfaisal, IAU)
- Full English ⇄ Arabic language toggle with right-to-left (RTL) support
- Sector filters and live search
- Responsive design for mobile, tablet, and desktop

## Running locally

It's a single self-contained `index.html`. Either:

- Open `index.html` directly in a browser, or
- Serve the folder: `python -m http.server 5500` then visit `http://localhost:5500`

## Notes

University recommendations are indicative and based on program reputation, accreditation, and specialization strength. Verify current programs and admission requirements with each university's official admissions office.
